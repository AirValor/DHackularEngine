class Window ;
Represents a window in which your game runs, can only have one instance.

static Window init (int w, int h, int d, int flags);
Initializes a window with parameter options, this should be the first function called. If window was already initialized, this function will just return the previous window. Enables init () to run if succeeded and returns the instance, and returns null otherwise

Params:
int w width of the window
int h height of the window
int d colorbyte depthcount
int flags bitsensitive integer for window settings

Returns:
The instance of Window.

See Also:
addFlags(int), resize(int, int), init ()
