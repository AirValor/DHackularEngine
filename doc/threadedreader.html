dhe.io.read.threadedreader

A file reader based on the actor threads of D to just send lines to your thread as quickly as it's available.

void startReader (Tid thread, string filename);
Initializes a new threadloop, but won't start throwing lines from the file until it gets a reply. It will send a message with only the reader Tid and will expect your own Tid back, after which it will start sending line strings.

Params:
Tid thread Receiver of the file content
string filename Name of the file to be read, preferrably with variable full path, and not relative path

Examples:
bool run = false;
startReader(thisTid, "../data/testfile.txt"); // TODO: implement build directory variable
receive(
	(Tid t){ t.send(thisTid); },
	(Variant any) { assert(false, "Stuff went wrong, got: "~any.toString()); }
	);
do{
	receive(
		(int code) { switch(code){
			case EndOfStream:
				writefln("Finished reading file");
				run = false;
				break;
			default:
				run = false;
				break;
		}},
		(string line) { writefln("Read line: %s", line); },
		(Variant any) { assert(false, "Stuff went wrong, got: "~any.toString()); }
		);
} while(run);
