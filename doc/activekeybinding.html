dhe.event.activekeybinding



void activate (Keybinding binding);
Set, or unset the currently active keybinding. Only one can be active at one time, so they will not collide with eachother.

Params:
Keybinding binding Keybindings that should react to the SDL events

void enable (bool value = true);
Enable or disable the active keybinding temporarily.
